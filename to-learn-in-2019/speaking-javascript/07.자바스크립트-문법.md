# 자바스크립트 문법

* 불리언
  * true
  * false

* 숫자
  * 1023
  * 7.851

* 문자열
  * 'hello'
  * "hello"

* 단순 객체

```

{
    firstName: "Jane",
    lastName: "Doe"
}
```

* 배열 

```

['apple', 'banana', 'cherry']
```

* 등호는 변수에 값을 할당한다.
* 3중 등호는 값을 비교한다.
* 표현식은 값을 생성하며 값이 있어야 할 곳에는 어디든 쓸 수 있다.
* 문은 어떤 행동을 하는 것을 의미한다.
